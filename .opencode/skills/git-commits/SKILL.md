# Conventional Commits

​
Format: `type(scope): description`
​
Types:

- feat: new feature
- fix: bug fix
- docs: documentation
- refactor: code restructuring
- test: adding tests
- chore: maintenance
  ​
  Example: `feat(auth): add OAuth2 login flow`
